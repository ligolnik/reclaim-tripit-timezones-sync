# Sync Travel Timezones on User Request

## Problem Description

Jbaruch just booked several new trips in TripIt and wants to make sure Reclaim.ai has the correct travel timezones set up. They message their coding assistant:

> "Hey, can you sync my TripIt travel timezones to Reclaim?"

The assistant should recognize this as a request to run the TripIt-to-Reclaim sync tool and execute it with the correct command, project directory, and timeout settings.

## Output Specification

Execute the sync by running the appropriate command. Do not write documentation — just run the sync.
